6.8 Service Test Management

Diagram Description

Service Test Management applications are focused on ensuring that the various services are working properly. Service test applications are part of both the fulfillment and the assurance process. In the fulfillment process, the service test is responsible for ensuring that the assigned service works as designed, while on the assurance side the service test applications are responsible for service problem isolation. As part of the testing process, these applications also interface with the trouble process, which can trigger an automatic test.
