Category: (4) eTOM Process Type

Process Identifier: 1.3.5.9.1

Original Process Identifier: 1.1.1.18.9.2

Maturity Level: 4

Description

Manage all requests (inbound and outbound) made by potential and existing customers

Extended Description

The purpose of this process is to manage all requests (inbound and outbound) made by potential and existing customers. It receives the request and either enables its originator to automatically fulfill it, or identifies and activates the opportune process to accomplish the request; it manages the status of the request and is able to provide status information at any moment in which the request is active; it formally closes the request when all related activities have been terminated.
