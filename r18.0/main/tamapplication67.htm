  • Description
  • Associations

Category: (3) TAM Application Type

Application Identifier: 5.3.1.6

Maturity Level: 3

Overview

Customer Order Validation is, like enrichment, widespread ordering functionality, that checks the validity of the order.

Functionality

Capability that ensures the qualified order is valid in any moment of order lifecycle. Forcing full validation in channels might increase complexity, so some channels do not provide fully validated orders. Validation ensures early fallout which is less costly than encountering errors in later stages of order handling. Validation checks the order not just at order entry point (like channels) but during the complete end to end order processing, usually as data become available.
