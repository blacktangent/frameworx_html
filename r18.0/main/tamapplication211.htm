Category: (2) TAM Application Type

Application Identifier: 9.2.8

Overview

Awards & Recognition provides the necessary guidelines and tools to properly and in a consistent way recognize and award employees.

Functionality

Awards & Recognition provides the following functionality:

  • Provides guidelines for appropriate employee recognition.

  • Provides guidelines for appropriate employee awards.

  • Provides tools and processes to generate awards such as monetary or item.
