  • Associations

Category: (2) TAM Application Type

Application Identifier: 5.10.1

Maturity Level: 4

Overview

Billing Inquiry applications provide capabilities to view a given customer’s invoice bill and supporting details in order to answer customer questions.

Functionality

The purpose of Bill Inquiry is to provide the capabilities to view customer bill and supporting details to answer a query on a bill. Query types may include the following:

  • Balance, statement

  • List all invoices

  • View exact bill image

  • List all charges per invoice

  • View unbilled charges

  • Generate bill on demand

  • View usage summary and details - View usage summary and details (billed and non-billed)
