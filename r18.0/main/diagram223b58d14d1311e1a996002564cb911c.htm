1.5.8.6 Close Resource Trouble Report Flow

Diagram Description

The objective of the Close Resource Trouble Report processes is to close a resource trouble report when the resource trouble has been resolved.
These processes monitor the status of all open resource trouble reports, and recognize that a resource trouble report is ready to be closed when the status is changed to cleared.
