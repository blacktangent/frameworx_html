Process Identifier: 1.6.7.3.2

Original Process Identifier:

Maturity Level: 4

Description

Communicate a Privacy Policy Profile attached to a PII to a Data Processor Party so that the Privacy of PII is still protected outside of the Data Controller and when with the Data Processor. Ensuring the end-to-end protection of the PII , ensuring privacy, is an essential requirement to avoid flaws in the data processing processes.

Extended Description

The Send Party Privacy Profile process manages the application of External Privacy Policy Rules when a PII must be sent to a Data Processor Party. It applies the External Rules, then transforms the existing Party Privacy Profile External Rules into Internal Rules, in order the Data Processor Party can import it into its Privacy Management.
