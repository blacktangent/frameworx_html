Overview

The BI Delivery Mechanism layer consists of the technologies utilized to present information to end user in the format requested and through the desired user interface.

Functionality

BI Delivery Mechanism application functionality includes the following:

  • Desktop Gadgets
  • Office Suites
  • Mobile
  • Disconnected
  • Portals/Business Mashups
  • Customer Facing BI
