Category: (2) TAM Application Type

Application Identifier: 5.18.5

Maturity Level: 4

Overview

Customer SLA Reporting applications provide the necessary functionality to report on Customer SLA performance as well as on open SLA issues.

Functionality

Customer SLA Reporting functionality includes:

  • Producing reports on open SLA issues
  • Producing reports on how the service provider is performing per the specified service levels.
