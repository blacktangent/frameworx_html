11.2 Fallout Management

Diagram Description

Fallout Management applications manage the manual steps within an order flow. These steps might be due to an error, an exception, or simply a planned manual step.

Fallout can come from order management applications within the customer, service, or resource layers

After the fallout/manual step is addressed, the order flow will presumably continue from that point within the ordering flow.
