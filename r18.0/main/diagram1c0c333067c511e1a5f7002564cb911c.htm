7.14 Usage Management

Diagram Description

Usage Management is the conduit for usage events from the network to various processes such as billing, legal compliance, and service assurance.  Network usage event records are collected/processed, edited, correlated, enriched, formatted and distributed to upstream systems.

The Usage Management function supports both session and event based charging in both batch and real-time. Usage event records that are found to have errors are processed per business rules.
