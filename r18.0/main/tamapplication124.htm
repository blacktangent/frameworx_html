Category: (2) TAM Application Type

Application Identifier: 6.3.4

Maturity Level: 4

Overview

The Service Order Design/Assign application performs end to end engineering design of service.

Functionality

The Service Design & Assign application leverages the corporate supply chain and asset inventory to assemble end to end design records. This design will be used by Service Order Decomposition for issuing work order or activation requests as appropriate to Resource Order Management.

The Service Design/Assign application functionality includes the design of the end to end solution as well as the assignment/procurement of network resources and customer premise equipment.
